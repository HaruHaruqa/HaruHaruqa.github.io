HANASU用VCVC音源マニュアル by Haruqa

■はじめに
	HANASU用VCVC音源マニュアルを手にとっていただきありがとうございます。
	
	
■利用規約
	このマニュアル
	（htmlファイル、画像、jsファイル（ただし、jqueryはMIT licenseに従います）、cssファイル、wavファイルなど）
	の再配布は禁止です。
	
	
■動作確認環境
	※javascriptを有効にしてください。
	FireFox
	Google Chrome
	
	
■ファイル構成
	・ReadMe.txt
		このファイルです。
		
	・index.html
		マニュアル本体です。通常はこのファイルをダブルクリックすることで、
		規定のブラウザが立ち上がり、マニュアルを閲覧することが出来ます。
		
	・css\
		マニュアル内で使用されているスタイルシートファイル群です。
		
	・img\
		マニュアル内で使用されている画像群です。
		
	・js\
		マニュアル内で使用されているjavascriptファイル群です。
